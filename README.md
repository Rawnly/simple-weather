# Weather
An npm commandline to get the [darksky](https://darksky.net) forecast.


# Installation
To install `weather-commandline` you must use a node package manager such as **npm** or **yarn**
```bash
  $ yarn global add weather-commandline
```

# Usage
```bash
  $ weather [options]

  -f --full 
  -r --restore
  -s --settings
```
